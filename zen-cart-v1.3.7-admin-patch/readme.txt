Zen Cart v1.3.7  Admin PATCH   Released June 30, 2007
====================================================
An admin vulnerability has been reported. The enclosed patch addresses the matter.
Please patch the enclosed files in order to protect your admin section further.
This patch supercedes and should be applied after any other patches already issued for this version.

FILES AFFECTED
==============
The files in this zip are:
/readme.txt (this file should not be uploaded to your site. All the rest should)
/admin/login.php
/includes/functions/sessions.php


DETAILS
=======

To combat a possible exploit vulnerability in Zen Cart, simply copy the 
enclosed files to your v1.3.7 website.

Remember, if you have renamed your admin folder, you will have to use *that* 
folder name when copying/uploading the /admin/ folder files.
